# 📊 Global Retail Revenue Intelligence & Forecasting System

## 🚀 Project Overview

This project is an end-to-end Business Intelligence and Revenue Optimization system built using Python and Power BI.

The goal was to analyze global retail sales data, identify revenue drivers, segment customers, and forecast future sales trends to support data-driven business decisions.

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn)
- Power BI (DAX, Forecasting, Interactive Dashboard)
- SQL-based data modeling approach
- GitHub for version control

---

## 📂 Dataset

Global Superstore dataset (2011–2014)

- 25,000+ Orders
- 4,800+ Customers
- Multiple markets (US, APAC, LATAM, EU)

---

## 📈 Key Analysis Performed

### 1️⃣ Revenue & Profit Analysis
- Market-wise sales comparison
- Category-level profitability
- Sub-category loss detection

### 2️⃣ Discount Impact Study
- High discounts significantly reduced profit margins
- Identified optimal discount ranges

### 3️⃣ Customer Segmentation (RFM Model)
- Recency, Frequency, Monetary scoring
- Segmented customers into:
  - Champions
  - Loyal Customers
  - At Risk
  - Potential

### 4️⃣ Shipping Performance Analysis
- Calculated average shipping days
- Identified operational improvement opportunities

### 5️⃣ Sales Forecasting
- Built time-series forecasting model
- Predicted next 3 months sales
- 95% confidence interval analysis

## 📊 Power BI Dashboard Pages

## 💡 Key Business Insights

- Technology category drives highest revenue.
- High discount rates (>30%) reduce overall profitability.
- 20% of customers contribute nearly 40% of revenue.
- Forecast indicates stable upward growth trend.
